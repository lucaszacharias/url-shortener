## 💻 Sobre o projeto

Neste projeto desenvolvi um aplicativo que serve como encurtador de URL, utilizando o Typescript e o Node.js e salvando os dados em uma base de dados na nuvem (MongoDB).
---

## ⚙️ Funcionalidades

- [x] Encurtar URLs

---

## 🚀 Como executar o projeto

Rodando o projeto:

- `npm install`
- `npm run build`
- `npm run dev`
